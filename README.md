# Portfólio Pessoal | Isari Design ✨

Este é o repositório do meu portfólio pessoal web, construído para unir e apresentar minhas duas paixões: o desenvolvimento Front-End e o design de marcas/identidade visual.

🔗 Acesse o projeto online aqui: https://isardrgs.github.io/Portfolio-Design/

## 💻 Sobre o Projeto

O objetivo deste projeto foi criar um espaço na web que refletisse a minha própria identidade visual como designer, garantindo uma experiência de usuário fluida em qualquer dispositivo. O site atua como uma vitrine para os meus principais projetos de design e UI/UX  com redirecionamento direto para o meu Behance.

## 🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido do zero, sem o uso de frameworks, para consolidar conhecimentos fundamentais da web:
* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização completa, uso de variáveis de cor, Flexbox e CSS Grid.
* **Responsividade:** Implementação de `@media queries` para garantir que o layout se adapte perfeitamente a celulares, tablets, desktops padrão e telas ultrawide (maiores que 16:9).

## 🌟 Principais Funcionalidades

* **Design Responsivo:** O site se redimensiona de forma inteligente, empilhando elementos no mobile e ajustando espaçamentos em telas gigantes.
* **Grid de Projetos:** Exibição organizada dos cases de sucesso com efeitos de *hover* dinâmicos (escala e sombra) para maior interatividade.
* **Tipografia Customizada:** Uso da família tipográfica *Space Grotesk* via Google Fonts para um visual moderno e limpo.
